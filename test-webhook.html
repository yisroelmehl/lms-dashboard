🔧 בדיקת Webhook קשר

⚠️ הוראות: החלף את ה-adddata (טוקן) בטוקן אמיתי מקישור תשלום קיים.
הטוקן נמצא בכתובת של דף התשלום: /pay/{TOKEN}

הגדרות בסיס

1. POST בטופס HTML (מומלץ - אין בעיית CORS)

שולח POST רגיל כמו שקשר עושה. יפתח את התשובה בחלון חדש.

2. GET - ניתוב חזרה (פתיחה בחלון)

מדמה את הכתובת שקשר מפנה אליה את המשתמש אחרי תשלום

3. פקודת curl (להרצה בטרמינל)

אם שום דבר לא עובד - העתק את הפקודה הזו והרץ בטרמינל
